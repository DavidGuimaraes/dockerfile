### Exemplo efcaz-test

Projeto efcaz-test é um projeto de testes funcionais do projeto efcaz.

* Necessidade de se montar um ambiente para executar os testes e depois derrubar o mesmo.
* Utilização das seguintes ferramentas:
    * Jenkins
    * Protractor
    * Maven
    * Liquibase
    * Docker
